煮一鍋香噴噴米飯的秘訣：

選米：買米時要選擇晶瑩剔透的新米。

洗米：輕輕地快速沖洗2至3次，不要用力搓洗。

加水量：一般米和水的比例為1.2杯水於內鍋。新米可略少於約1.1杯水，舊米要多些約1.3杯水。

浸泡：煮飯前浸泡，夏天約浸泡半小時，冬天約1小時。

燜飯：煮飯完成後需燜10~15分鐘，且中途不能打開鍋蓋。

鬆飯：食用前，將飯翻鬆，讓多餘的水份蒸發掉，米飯會更香Q可口。
