3.2. Navigation

Site navigation is found in #sidebar div inside #main div. To create main menu items, add list elements to #navigation div. Inside the list element's anchor tag you can edit the number of the menu (.menu-number span, remove if you would not like to use it) and the menu description (.menu-desc span). You can add sub menu items to the main menu items adding an ul tag with list elements inside. You also have to add the parent class to the parent li elements. Adding current class to list elements makes the menu item selected/highlighted.

Notes

Navigation works also without javascript. Without js all submenus are open so you can click to each menu item.

If you remove .menu-number span you should also edit the styles.css file to look the submenu item properly. Change #navigation ul li a {...} css selector's left padding value from 70px to 10px.

6.2. Latest tweets

Plugin options

  • user: twitter screen name (string) // default value: granthweb
  • count : number of tweets (integer) // default value: 2
  • showRetweets: show or not to show retweetes (boolean) // default value: true

Notes

plugin name: gwLatestTweets().
Plugin code and settings are located in script.js file.

Call the plugin

/* call latest tweets plugin */
var $tweets=$body.find(".tweets");
if ($tweets.length) { $tweets.gwLatestTweets(); };

Call the plugin with custom options

/* call latest tweets plugin */
var $tweets=$body.find(".tweets");
if ($tweets.length) { 
	$tweets.gwLatestTweets({
		'user' : 'my_twitter_name',
		'count' : 3,
		'showRetweets' : false
	});
};

6.3. Preload images

Images can be preloaded with a custom plugin. You should select the images you would like preload and call the plugin.

Plugin options

  • alwaysShow: show or not to show the effect (boolean)
  • delay: image animation delay (integer)
  • fadeInDuration: fade in effect duration (boolean)
  • checkInterval: loaded status check time intarval (integer)

If you would like to preload images which are not loaded and cached yet, you should set alwaysShow option to false. Setting alwaysShow param to true forces effect for even loaded/cached images. Smaller duration value makes animataion faster. Setting delay makes pause/delay in fadein effect in a set of images. Images with higher index show later.

Notes

plugin name: gwPreloader().
Plugin code and settings are located in script.js file.

Images cannot be truely preloaded via javascript. This plugin like every other preloader plugin is fake preloader, only shows a preloader effect while images are loading.

Call the plugin

/* show preload effect on images if they not in cache - call image preloader plugin */
$body.find('.image-frame img, .portfolio-items img').gwPreloader();

Call the plugin with custom options

/* show preload effect on images if they not in cache - call image preloader plugin */
$body.find('.portfolio-items img').gwPreloader({ 
	'fadeInDuration' : 350,
	'delay' : 50,
	'alwaysShow' : true,
	'checkInterval' : 200
});

6.4. Slider plugin

All sliders in this template use this plugin. You can create image slider or html content slider using gwSlider.

Plugin options

  • startIndex: index of the first slide (integer) // default value: 0
  • showControls: show or not show control arrows (boolean) // default value: true
  • showPagination: show or not show pagination dots (boolean) // default value: true
  • autoPlay: autoplay slider (boolean) // default value: true
  • autoPlayInt: pause time in ms between two slides when autoplay option is on (integer) // default value: true
  • autoPlaypauseOnHover: pause or not pause when mouse is over slider (boolean) // default value: true
  • easing: an easing type (requires jQuery easing plugin) (string/boolen), set this value to false to disable easing // default value: false
  • animSpeed: slide animation speed in ms (integer) // default value: 700
  • heightSpeed: height resize animation speed in ms (integer) // default value: 400
  • slideDelay: delay in ms between two slides (integer) // default value: 0
  • loop: makes the slider infinite if its true (integer) // default value: false

This template uses this plugin for widget slider and content sliders. Content sliders are used in blog posts, project posts and a delayed testimonials slider in about.html.

Notes

plugin name: gwSlider().
Plugin code and settings are located in script.js file.

Call the plugin

As you can see below this plugin is called with different options for sliders.

/* load widget slider - call plugin */
$body.find(".widget-slider").gwSlider({'animSpeed':500, 'showPagination':false, 'autoPlay':true, 'loop':true});
		
/* load testimonials slider - call plugin */
$body.find(".testimonials-slider").gwSlider({'animSpeed':700, 'slideDelay':80});
					
/* load project & post slider - call plugin */
$body.find(".post-slider, .project-slider").gwSlider({'animSpeed':700});
